## Áttekintés
Az előző megbeszéléshez képest jelentős előrelépés történt az API-k és a frontendes adatműködés összekötésében.  
Több kritikus hibajavítás és új funkció is megvalósult, a rendszer már közelít a bemutatható állapothoz.

## Fejlesztések

### Backend
- Messages API endpoint létrehozása (üzenetküldés és -fogadás alapjai).
- API endpoint a jelenlegi felhasználó jegyéhez és szerepköréhez.
- API endpoint a kurzusok lekérésére a jelenlegi felhasználó osztálya / tanszaka alapján.
- Course API frissítése, hogy a bejelentkezett felhasználó departmentjéhez igazodjon.
- Application.properties refaktorálása az egyszerűbb használat érdekében.
- PostgreSQL kapcsolat javítása, CORS konfiguráció javítása.
- Frontend–backend kapcsolat javítása, sikeres bejelentkezés biztosítása (login most már működik).

### Frontend
- Logout button, link hover és új dummy.js létrehozása a tesztekhez.
- Üzenetek oldal bugfix (üzenetek.html hibajavítás).
- Órarend adatok betöltése és megjelenítése frissítve.
- Órarend aktualizálása, vizuális módosítások.
- Megnyíló modulok formázása (modális/slide elemek).
- Kereső formázása, nagyobb használhatóság.
- Dokumentáció frissítése és refaktorálása.

### Egyéb
- Build folyamat frissítése (pom.xml módosítás, application.properties refaktor).
- Mai meeting riport rögzítése a dokumentációba.
